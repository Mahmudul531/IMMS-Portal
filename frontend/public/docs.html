IMMS Intelligent Maintenance Management System

Comprehensive Platform User Manual & Administrator Guide

Overview & Authentication

The IMMS Portal is a full-scale Enterprise spatial registry and asset lifecycle management system.

Registration & Login

  • Login System: Secure login requires an authorized Username and Password. Access is strictly audited.
  • System Roles: The system evaluates you as ADMIN, ENGINEER, TECHNICIAN, or VENDOR. Features and data views are heavily restricted based on your access tier.
  • New Users: Only an Administrator can create and assign roles to new users via the "User Access" hub.

1. Dashboard Analytics

Upon successful authentication, the Dashboard provides an immediate analytical summary of the platform's health.

  • Metrics: Displays total active properties, aggregate asset count, and total open work volumes.
  • Real-Time Spatial Map: Internally aggregates every single Property coordinate and displays a customized map layer (powered by Leaflet GIS capabilities). Hovering/clicking map pins identifies precise geographical properties instantly.

2. Property Management

Properties act as the root spatial zones where all assets are housed.

  • Creation: Admins and Engineers can register new properties. It requires an exact address and exact Latitude/Longitude coordinates (used for the global map registry).
  • GIS Searching: Easily search locations automatically via integrated Geolocation services.
  • Property Detail View: Each property holds an independent gallery of structural photographs uploaded securely to the cloud.

3. Assets & Inventory Mapping

Assets are the core mechanical, structural, or electronic items assigned identically to Properties.

  • Asset Addition: Assets are recorded with strict purchase financial data, categorical types, and specific supplier data. Adding future dates is systematically restricted.
  • Financial Depreciation: When viewing a specific Asset internally, the system initiates mathematical calculations to derive real-time present value. Depreciation only takes effect strictly starting AFTER the first year of ownership.
  • Asset Gallery & Invoices: Users can natively attach image galleries of the mechanical asset natively, and securely attach raw PDF Invoices.
  • Activity Timeline: Generates an automatic visual chain of custody tracing when the asset was created and exactly where it has been transferred.
  • Exporting Data: Authorized users can instantly export complete historical chains to secure PDF documents or raw XLS spreadsheets.

4. Asset Setup (Preferences)

A specialized configuration matrix designed solely for Administrators.

  • Administrators dynamically create, delete, and manage universally selectable categories (e.g., HVAC, Electrical, Plumbing) and system Types.

5. Inter-Property Asset Transfers

When physical assets are moved between regional locations, it must be digitally authorized.

  • A streamlined tracking matrix allows Engineers to safely un-link an asset from "Property A" and officially register it to "Property B".
  • Mandatory digital signatures and notes are applied to create a foolproof historical ledger.

6. Work Orders & Vendor Application Bidding

The central hub for resolving mechanical issues or initiating operational maintenance tasks.

  • Deploy Duty: Administrators create targeted Work Orders and map them uniquely to a root Property and sub-selected Asset.
  • Vendor Bidding: External Third-Party Vendors log in directly to review "Pending" tasks. A Vendor can securely evaluate the scope and submit a competitive numerical bid (BDT ৳) for the job.
  • Awarding Contracts: Administrators review submitted bids by multiple Vendors internally. Once a bid is approved, the vendor is structurally Locked in and the duty shifts to ACTIVE/ASSIGNED mode.

7. User Access Matrix

Complete control interface over who holds organizational access.

  • Creation Matrix: Admins dynamically register operational staff.
  • Power Toggles: Admins possess instantaneous override locks to temporally Suspend or fully Deactivate accounts without destroying historical logs connected to that user.
System Notification: PDF exports, XLS logs, and Cloud CDN image deliveries are directly powered by our secure regional infrastructure. Unauthorized downloading attempts or unauthorized API calls will instantly return 401 Unauthorized server traps.
