Core domain entities mapped using Spring Data JPA and Hibernate. Note: Spatial support uses locLat/locLon strings instead of PostGIS geom, and IDs are Longs, not UUIDs.

Column Name	Data Type	Constraints/Description
id	Long	PK, Auto-increment (Identity)
username	String	Unique, Not Null
password	String	Not Null
role	String	Not Null (ADMIN, ENGINEER, TECHNICIAN, ACCOUNTS, CONTRACTOR)
email	String	
active	Boolean	Default True
inactiveNote	String	
fullName	String	
phone	String	
dob	LocalDate	
gender	String	
nidOrPassport	String	
department	String	
designation	String	
propertyId	Long	FK to Property
permissionGroupId	Long	FK to PermissionGroup
createdAt	LocalDate	Creation Timestamp

Entity: PermissionGroup
Column Name	Data Type	Constraints/Description
id	Long	PK, Auto-increment
name	String	Unique, Not Null
permissions	String	Text (Comma-separated roles)
createdAt	LocalDate	Creation Timestamp

Column Name	Data Type	Constraints/Description
